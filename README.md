# simple-social
Let's get as vain as possible - username, followers, and nothing else

## Features
* Back end 
	* Get users
	* Create users
	* bcrypt passwords
	* Users can follow other users
	* Delete users

* Front end
	* List all users and followers
	* Add and delete users
